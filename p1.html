01 : Probablistic Computing

Project Title:
Design and Implementation of an Energy-Efficient Hardware for Probabilistic Computing

Abstract:
Why Probabilistic Computing?
Probabilistic computing paradigms provide a bridge between classical and quantum computing. Which is constructed using probabilistic bits and based on Ising machines and Boltzmann machines. We can emulate the quantum phenomena using currently available devices (CMOS) by applying probabilistic Spin Logic. It worked perfectly at room temperature and uses the time window until maturing quantum technology.

Applications in Combinatorial Optimization, Stochastic Circuits, Bayesian Inference Network, Invertible Boolean Logic, Machine Learning, Quantum Emulation.

Goal: To design energy-efficient hardware accelerator for probabilistic computing.

Language & Tools Use:
Verilog Language, Xilinx (Vivado), FPGA, ASIC (Cadence)
