003 : CAN Bus

Project Title:
Design and Implementation of an Embedded System for CAN Bus to Bluetooth Data Logger

Abstract:
A Controller Area Network (CAN Bus) is a robust vehicle bus standard designed to allow microcontrollers and devices to communicate with each other's applications without a host computer.

Our main objectives are the following:
1. Sending simulated vehicle data through Bluetooth (CAN Master 1).
2. Implement a Bluetooth interfaced CAN data logger/ bus monitor.
3. Shows real-time traffic on CAN bus on a Bluetooth receiver (through a mobile App).
4. Store the bus data on the Bluetooth receiver for detailed analysis.

The CAN bus traffic is simulated using two TIVA boards and two CAN transceivers (MCP2551). Where Master 1 will send the vehicle simulated data through Bluetooth and Master 2 will flow some messages continuously. Successfully, Done in Embedded System Mini Course Project.

Language & Tools Use:
C, TIVA Board, Code Composer Studio, CAN transceivers, Bluetooth Module
