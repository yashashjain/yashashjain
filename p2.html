002 : CORDIC Algorithm

Project Title:
FPGA Implementation of Sine Waveform Generator using 12 Stage Pipelined CORDIC Algorithm

Abstract:
The sine and cosine waves are the fundamental building blocks of communication and signal processing. With the development of handheld mobile computing devices, the requirement for high-performance compact processors is increasing.

So, FPGA Implementation of Sine Waveform Generator Based on 12 Stages Pipelined CORDIC (COordinate Rotation DIgital Computer) Algorithm is done in Digital System Design with FPGA mini-course project at IISc.

Language & Tools Use:
VHDL Language, Xilinx (Vivado), FPGA (Basys3 Board)
